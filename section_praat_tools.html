1. Praat plug-ins

In this section you will find some plug-ins I have written for Praat. They are focus on making easy the linguistic annotation of corpora and database management.

If you are not familiar with plug-ins, do not be afraid! You should know that these tools are user-friendly and add new features to Praat (you will see some kind of new buttons and menus!).

1.1 How to install a plug-in?

To install a plug-in, (1) pick one project from this website, then download and unzip it. Now, you should have the Plug-in folder. Then, (2) find the Praat preference folder. This comes by default when you use Praat for the first time. Depending on you operating system, these could be the locations:

  • In Linux, go to /home/user_name/.praat-dir
  • In Mac, go to /Users/user_name/Library/Preferences/Praat Prefs/
  • In Windows, go to C:\Users\user_name\Praat
Now, (3) drop the Plug-in folder inside the Preferences folder. Finally, (4) open Praat. If Praat was opened at the time you installed the plug-in, you will not see any change. So, close and open it again.

2. Scripts and procedures

In this website, you can also find scripts templates that tackle some common tasks like looping through TextGrids inside a folder or editing audio files. There are also procedures I have written that you could include in your scripts.

PyPhon

A set of tools written in Python to help with phonology analysis

Phonetic report

Example 1: Make a phonetic report and write it to report.txt in a word list file 
    
    python3 cli-mkreport.py -i /home/Desktop/wordList.txt -printf "wWsl" > report.txt

PhonStats

Example 1: : Get all segments, frequency and check if they belong to a segment inventory in a word list file 
    
    python3 cli-phonstat.py -i /home/Desktop/wordList.txt -fe

Replace

Example 1: Replace SAMPA "1" to "ɨ" in a word list file 
    
    python3 cli-replace.py -i /home/Desktop/wordList.txt -r "1" "ɨ"

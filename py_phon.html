PyPhon

A script written in Python to find and report minimal pairs from arrays or a plain text.

Example 1: Get minimal pairs from an array 
    
    words = ['peso', 'perro', 'paso', 'piso', 'a', 'karo', 'karro']
    test = MinimalPair()
    test.set_characters_as_one_unit(['sh', 'ch', 'rr'])
    test.compare_all(words)
    test.show()
Example 2: Get minimal pairs from a plain text 
    
    textPath = '/home/rolando/Documents/wordList.txt'    
    test = Corpus(textPath)
    test.read(clear = False)
    test.set_characters_as_one_unit(['sh', 'ch', 'rr'])
    test.compare_all(list(test.dict_words.keys()))
    test.show()
    test.save()

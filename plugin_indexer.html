Finder v2.0.0

This plug-in searches for those intervals or points in your TextGrid files that contain a specific label. When a search is done, you can view the results in the TextGridEditor. Also, you can extract those matched parts and save them as new files. More advanced users can use scripts to do other tasks.

In this tutorial, I will show you how to use this plug-in.

Getting started

How to use it?

Follow these three steps in order:

1. Create an index

Go to Create index... in the Do menu. In the dialogue box, complete the TextGrid folder field with the path where your TextGrid files are stored. Then, click on Ok or Apply. By doing so, Finder will collect all the labels from the point and interval tiers of your TextGrid files.
When the process is done, Praat will print a summary in the Info window. It will show you all the tiers from your TextGrids and the sum of their labels. Tiers which does not have any label (in the TextGrids) will not be included in this report.
Adionally, you will see an index Table in the Object window. This table is only for informative purposes. Feel free to remove it.

2. Search a text

Now, use the Search command of the main menu to look for specific labels in your TextGrid files.
In the dialogue box, pick the tier level on which your search will be done. Then, complete the Search for and Mode fields. For example, if you want to match all the labels that start with "a" in a specific tier, complete the Search for with the label "a" and pick the starts with mode. Then press in Ok or Apply.
At the end, Praat will report the results in the Info window. You will see the Total number of occurrences and, if your search is possitive, a list with non-duplicated labels called Search Summary.
